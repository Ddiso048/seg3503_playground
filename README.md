## Java - Comment exécuter

1. Aller dans le dossier newmath_java
2. Compiler le code :
bin/compile
3. Exécuter le code :
bin/run
4. Exécuter les tests :
bin/test

## Elixir - Comment exécuter

1. Aller dans le dossier newmath_ex
2. Exécuter le code :
mix run -e "IO.inspect NewmathEx.div(10, 4)"
3. Exécuter les tests :
mix test
